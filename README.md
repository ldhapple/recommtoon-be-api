# 소개


> MBTI로 추천받는 나만의 웹툰

RecommToon은 MBTI, 성별, 나이, 평가한 웹툰 정보를 가지고 개인별로 딱 맞는 웹툰을 추천하기 위한 서비스 입니다. <br>
본 웹툰들에 평가하고, 다른 MBTI, 성별, 나이에서는 어떤 웹툰이 인기있는지 확인하며 조금 더 웹툰에 대한 관심을 갖고 다양한 웹툰들을 볼 수 있도록 하는 것이 목표입니다.
